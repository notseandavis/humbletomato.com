What Is This?

A collection of ideas about how software development changes when AI writes most of the code. Some pages explore why things work (or don't). Others describe how to do it. Start wherever looks interesting.

Vibe vs Spec

When should you chat with AI versus write structured specs? The tradeoffs aren't obvious.

🎯

The Solution Space

Every prompt maps to a space of possible outputs. Specs narrow that space. Vague prompts don't.

📞

The Telephone Game

Information degrades every time it passes through a layer. LLM chains make this worse.

🪞

The Abstraction Paradox

Hiding complexity from the AI doesn't remove it. It just moves the debugging later.
